jupyter可以启动,但是无法连接kernel的解决方案

问题集 jupyter
一个问题
Publish Date:   2021-05-23

问题描述

使用pip安装jupyter,可以打开web,但是无法连接kernel,无法使用.

错误示例:

查看tornado,发现是6.1的版本, 依据网络上查到的降低tornado版本, 但是没有效果!

解决方案

实际的解决方案是,降低nbconvert的版本, 直接使用 pip install nbconvert==5.6.1 即可.
