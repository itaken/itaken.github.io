<mark>问题描述</mark>

>因为不小心将 **软件中心** 删除了, 重新安装的时候,无法正常安装.

```bash
$ sudo apt install ubuntu-software

因为系统无法达到您要求的状态造成的。该版本中可能会有一些您需要的软件
包尚未被创建或是它们已被从新到(Incoming)目录移出。
```

<mark>解决方法</mark>

安装**指定版本**的依赖包, 使用 `sudo apt-get install package=version` 的命令即可.
