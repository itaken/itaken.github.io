1. 当需要使用 `mb_*` 函数时候, 例如: **mb_strlen()**, 则需要安装`mbstring`扩展

    ```bash
    $ sudo apt install php7.0-mbstring
    ```

1. 当需要在php中进行高精度的运算, 需要使用**bc函数**, 则需要安装**bcmath**扩展

    ```bash
    $ sudo apt install php7.0-bcmath
    ```
