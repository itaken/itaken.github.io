1. 安装`dva-cli`工具.
    ```bash
    $ cnpm i dva-cli -g
    ```

2. 使用`dva new` 创建项目
    ```bash
    $ dva new dva-demo
    $ cd dva-demo
    ```

3. 运行项目
    ```bash
    $ npm start
    ```

    ![dva demo]({{ site.url }}/assets/images/201703/21-01.png)

    浏览器会自动开启，并打开 `http://localhost:8000` 页面,然后就可以修改你的dva项目了.
