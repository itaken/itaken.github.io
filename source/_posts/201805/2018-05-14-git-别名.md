查看别名列表

```
user@DESKTOP-K146JU9 path/to/world         
> git config --get-regexp alias                          
alias.ck checkout                                        
alias.br branch                                          
alias.cm commit                                          
alias.st status                                          
alias.pl pull                                            
alias.p push                                             
alias.m merge                                            
alias.s stash                                            
alias.r stash apply                                      
```

修改别名

```
user@DESKTOP-K146JU9 path/to/world         
> git config --global alias.cm "commit -am"              

user@DESKTOP-K146JU9 path/to/world         
> git config --get-regexp alias                          
alias.ck checkout                                        
alias.br branch                                          
alias.cm commit -am                                      
alias.st status                                          
alias.pl pull                                            
alias.p push                                             
alias.m merge                                            
alias.s stash                                            
alias.r stash apply                                      
```
