>环境配置
```
ubuntu 17.04
```

<mark>问题描述</mark>

如何静默的运行程序?


<mark>解决方法</mark>

只要在程序的后面添加`&`即可.

```bash
~/Softwares/pycharm-community-2017.2.3/bin  $ sh pycharm.sh &
[1] 2546
```

运行结束后,会提示如下信息:

```bash
~/Softwares/pycharm-community-2017.2.3/bin  $
[1]  + 2546 done       sh pycharm.sh
```
