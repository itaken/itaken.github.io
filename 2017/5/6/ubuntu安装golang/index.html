ubuntu 安装golang

ubuntu golang
开发日常
Publish Date:   2017-05-06

  1. 直接下载golang安装包, 或者使用wget或aria2c下载安装包.

    $ aria2c -c https://storage.googleapis.com/golang/go1.8.1.linux-amd64.tar.gz

    aria2c 可以使用sudo apt-get install aria2命令直接安装, 国内可以使用代理下载

    $ aria2c -c --http-proxy=xxx https://storage.googleapis.com/golang/go1.8.1.linux-amd64.tar.gz

  2. 直接解压到/usr/local/ 目录下

    $ sudo tar -C /usr/local -xzf go1.8.1.linux-amd64.tar.gz

  3. 在~/.profile文件 PATH 中,添加/usr/local/go/bin路径

    查看 version,确定安装成功

    $ go version
go version go1.8.1 linux/amd64
